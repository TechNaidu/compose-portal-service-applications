📖 About This Project

The Compose Portal Application is a multi-module Spring Boot microservices project designed for learning and demonstrating cloud-native application development.

🎯 Project Goals

  • Build a multi-module Maven project with independent microservices
  • Each service runs independently with its own port and static data
  • Static pages (Thymeleaf) + REST APIs in every service
  • Ready for Docker containerization and Kubernetes deployment
  • Designed for AWS cloud hosting (ECS, EKS, EC2)
  • Easy to add new modules/services to the project

🏗️ Architecture

Services

  • Gateway Service (Port 8080) — Central dashboard, service registry, health checks
  • User Service (Port 8081) — User management with CRUD operations
  • Product Service (Port 8082) — Product catalog and inventory
  • Order Service (Port 8083) — Order tracking and management

Project Structure

compose-portal-application/ (Parent POM)

🚀 How to Run

Build All Modules

mvn clean install

Run Individual Services

cd user-service && mvn spring-boot:run
cd product-service && mvn spring-boot:run
cd order-service && mvn spring-boot:run
cd gateway-service && mvn spring-boot:run

Run with Docker Compose

docker-compose up --build
